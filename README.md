# Object Segregation Robotic Arm

Segregating Nuts, Bolts, and Screws using a Dobot Magician Robot Arm, yolo, and realsense camera

## Installation
- Clone the repository
- Install the python packages using the requirements.txt file

## Calibration
- Run the calibration/calibration_simplified.py program to find the transformation matrix from Camera to Base

## Run the program
- Run the main/main.py program to run the segregation program
